# Product Roadmap

## Q2 2026: Performance & Depth
- **Enhanced Box Scores**: Add fielding stats (errors, putouts, assists).
- **Spray Charts**: Visual representation of hits for professional scouting.
- **Advanced Standings**: Strength of Schedule (SoS) and Tie-breaker rules visualization.
- **Pitch Tracking**: Velocity and pitch type tracking in the Box Score.

## Q3 2026: Connectivity & Reach
- **Public Viewer Mode**: Read-only public link for fans to follow live scores.
- **REST API**: External access for media outlets to fetch results.
- **Live Stream Overlay**: OBS-compatible browser source for scoreboards.
- **Mobile App (iOS/Android)**: Native application for offline scoring and push notifications.

## Q4 2026: AI & Insights
- **Game Forecasting**: Matchup analysis based on historical player data.
- **Automated Recaps**: AI-generated game summaries based on box scores.
- **Video Integration**: Link specific plays/at-bats to video timestamps.

## Infrastructure
- **Offline Sync**: PWA support for environments with unstable internet.
- **Multi-League Support**: Manage multiple concurrent tournaments.
